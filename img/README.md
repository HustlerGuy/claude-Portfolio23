# Zrzuty ekranu do realizacji

Wgraj tutaj pliki zrzutów ekranu. Kafle na stronie **Realizacje** pokażą je
automatycznie po wczytaniu (mechanizm `onload`). Nazwy plików muszą być dokładnie takie:

## Realizacja 001 — Google Ads (firma sprzątająca)
| Plik | Co przedstawia |
|------|----------------|
| `gads-marzec-2026.png` | Przegląd miesiąca (kafel główny) — 431 kliknięć, 45 konwersji, 2,15 tys. zł, 47,71 zł/konw. + wykres |
| `gads-czerwiec-2026.png` | Najlepszy dzień (mały kafel) — 12 kliknięć, 5 konwersji, 75,01 zł, 15,00 zł/konw. |
| `gads-przeglad.png` | Panel przeglądu kampanii (mały kafel) — zbliżenie |

Obsługiwane formaty: `.png`, `.jpg`. Jeśli użyjesz `.jpg`, zmień też rozszerzenie
w `realizacje/index.html` (atrybut `src` w znaczniku `<img>`).

Zalecana szerokość: min. 1200 px. Zrzuty są kadrowane od lewego górnego rogu
(`object-position: top left`).
